I have started this project as the simple commands translator to make it easier for me to configure devices that are not using Cisco like commands on which im used to. 
After some time working on it, i have decided to integrate AI in it for the features like importing whole configuration files to it, translating them and sending the files to the device that need to be configured.
As the project growed i have included one more feature and that is the auto configuration on devices trough AI model and from this release it should also be possible to configure some network stuff on Linux, Unix and Bsd. (Im using the word "should" because is not yet tested). 
You should be able to ask for a configuration file with the desired values like subnet mask, ip range etc. Check the generated file after generating it and send it to device of choice.
The project is still in testing phase so not every option is yet tested (like this above), so to make it better i have posted it here for anyone to use it and contribute to the project as they know: with testing, 
adding more functions, submiting new ideas, etc.
